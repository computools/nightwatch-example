# E2E with Nightwatch.js 

Project template and simple test examples

## Installation

Install required dependencies
including Selenium Server and `chromedriver`:
```
npm install
```

## Execution

E2E:
```
npm run test
```

VRT:
```
npm run test:vrt
```

## Organizing 

Tags:
```
'@tags': ['example'],
```

Ignore:
```
'@disabled': true
```
